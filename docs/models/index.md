## 🎯 Escolhendo o Modelo Certo

### Para Desenvolvimento
- **Claude**: Análise de código, debugging, refatoração
- **GPT-5**: Frontend, prototipagem, documentação
- **Gemini**: Análise de imagens, diagramas de arquitetura

### Para Pesquisa
- **Perplexity**: Research acadêmico, citações
- **Grok-4**: Informações atuais, tendências
- **Gemini**: Análise multimodal de dados

### Para Escrita
- **Claude**: Análise e estruturação de conteúdo
- **GPT-5**: Escrita criativa, marketing
- **Perplexity**: Artigos com fontes e referências

## 🔧 Configurações Comuns

Cada modelo inclui:

- **raycast-snippets-pt-br.md**: Snippets em português para Raycast
- **Arquivos específicos**: Templates, regras e configurações únicas
