    Modelos de IA

    Esta seção contém configurações específicas, snippets e guias para diferentes modelos de linguagem artificial.

    🤖 Modelos Disponíveis

    • Claude (Anthropic)

      Especializado em raciocínio analítico, programação e análise de código

      Características: - Excelente para análise de código - Forte em raciocínio lógico - Ótimo para debugging e refatoração

      Explorar Claude

    • Gemini (Google)

      Modelo multimodal com capacidades avançadas de pesquisa

      Características: - Processamento multimodal (texto, imagem, vídeo) - Integração com serviços Google - Forte em pesquisa e análise de dados

      Explorar Gemini

    • :material-openai: GPT-5 (OpenAI)

      Modelo versátil focado em criatividade e conversação

      Características: - Excelente para escrita criativa - Forte em desenvolvimento frontend - Ótimo para brainstorming e ideação

      Explorar GPT-5

    • Grok-4 (xAI)

      Modelo dinâmico com acesso a informações em tempo real

      Características: - Acesso a dados em tempo real - Tom conversacional e humor - Integração com X (Twitter)

      Explorar Grok-4

    • Perplexity AI

      Especializado em pesquisa com citações e fontes

      Características: - Pesquisa com citações automáticas - Acesso a fontes confiáveis - Excelente para research acadêmico

      Explorar Perplexity

    🎯 Escolhendo o Modelo Certo

    Para Desenvolvimento

    • Claude: Análise de código, debugging, refatoração
    • GPT-5: Frontend, prototipagem, documentação
    • Gemini: Análise de imagens, diagramas de arquitetura

    Para Pesquisa

    • Perplexity: Research acadêmico, citações
    • Grok-4: Informações atuais, tendências
    • Gemini: Análise multimodal de dados

    Para Escrita

    • Claude: Análise e estruturação de conteúdo
    • GPT-5: Escrita criativa, marketing
    • Perplexity: Artigos com fontes e referências

    🔧 Configurações Comuns

    Cada modelo inclui:

    • README.md: Guia específico do modelo
    • api-config.json: Configurações otimizadas da API
    • raycast-snippets-pt-br.md: Snippets em português para Raycast
    • Arquivos específicos: Templates, regras e configurações únicas

    🚀 Próximos Passos

    1. Escolha um modelo baseado no seu caso de uso
    2. Leia o README específico do modelo
    3. Configure a API usando os parâmetros otimizados
    4. Instale os snippets no Raycast para acesso rápido
    5. Explore os templates e exemplos práticos

    Dica de Produtividade

    Use os códigos de 3 letras nos snippets Raycast para acesso rápido. Por exemplo: dev para desenvolvimento, per para pesquisa, age para agentes.
